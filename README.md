# Семантический поисковик (Semantic Search Engine)

Система контекстного поиска по русскоязычным текстам, собранным из Википедии. Программа ищет абзацы по смыслу, а не просто по точному совпадению слов.

## Технологии и алгоритмы
* **Wikipedia API**: автоматизированный сбор данных и формирование динамического корпуса текстов.
* **TF-IDF (Term Frequency-Inverse Document Frequency)**: векторизация текстов, оценка важности слова в контексте отдельного абзаца и всего корпуса.
* **PyMorphy3 & NLTK**: пайплайн предобработки, включающий токенизацию, лемматизацию и фильтрацию стоп-слов русского языка.
* **Косинусное сходство (Cosine Similarity)**: метрика ранжирования результатов поиска.

## Математическая модель
Для поиска наиболее релевантных текстов алгоритм вычисляет косинус угла между вектором запроса и векторами документов в TF-IDF пространстве:
$$similarity(A, B) = \frac{A \cdot B}{||A|| \times ||B||}$$
Где $A$ — вектор пользовательского запроса, а $B$ — вектор абзаца из корпуса. Чем ближе значение к единице, тем выше смысловое сходство.

## Структура проекта
* `src/indexer.py` — модуль для парсинга Википедии и сохранения чистого корпуса текстов.
* `src/search.py` — основной движок поиска, векторизации и ранжирования.
* `requirements.txt` — список необходимых зависимостей.
* `data/` — директория для хранения корпуса текстов.

   cd <название_папки_проекта>
   
2. **Установите зависимости:**
    ```bash
    pip install -r requirements.txt

3. **Соберите корпус текстов (скачивание из Википедии):**
    ```bash
    python3 src/indexer.py

4. **Запустите поисковик:**
    ```bash
    python3 src/search.py
